*******************************************Info***************************************************

The following Project is a prototype build up of a typical Mission Control for a Space Mission 
illustrating the concept of Internet of Things.

An Android Application which can be considered as a Sensor Subsytem, sends Data to a remote server
(here : Thingspeak)
A MATLAB source code retrieves data from the server and displays it on the GUI.


**********************************Steps to run the program***************************************

1. Run Matlab

2. Navigate to directory that contains
     -MissionControlGUI.m
     -MissionControlGUI.fig

3. Run Program

4. GUI Opens

5. Click on 'start' to receive data from public server

6. Click on 'refresh' to refresh data on GUI.

7. Click on 'Exit' to exit the GUI.

Enjoy Learning.!
